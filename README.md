# trab-final-alp
Programa sobre Consoles que foi criado com base no CRUD que possui um menu que o usuário pode escolher entre: criar, listar, editar, e deletar um **Console**, sendo os mesmos sendo armazenados em um vetor, além de também o vetor ja ser inicializado com 4 consoles e os objetos possuirem os metodos get e set.
